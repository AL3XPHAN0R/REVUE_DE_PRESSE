# Digest Voice & Format (Layer 3 — Reference)

## Tone
- Professional but accessible — podcast educator style
- Make complex insurance topics approachable for non-experts
- French-first (Quebec market); English sources are welcome, flag them with 🇬🇧
- One punchy insight sentence per article — no more

## Telegram Markdown format (parse_mode=Markdown)

```
📰 *Revue de presse — [JOUR FR] [DD MMM YYYY]*

[EMOJI] *[Titre]*
_[Source]_ · [🇫🇷 ou 🇬🇧]
💡 [1 phrase : pertinence podcast ou angle PME]
🔗 [URL]

—

[répéter pour chaque article]

_[N] articles sélectionnés_
_Sources : r/smallbusiness · r/Entrepreneur · r/PersonalFinanceCanada · C'est pas mon idée · Repreneuriat Québec · Newswire Canada · Financial Post_
```

## Emoji guide
| Emoji | Use for |
|-------|---------|
| 🔒 | assurance / protection |
| 💰 | finances / PME |
| 🤖 | fintech / insurtech |
| 📊 | statistiques / données |
| 🏛️ | réglementation (AMF, CIRO, Chambre) |
| 💡 | idée podcast / angle créatif |
| ⚠️ | risque / mise en garde |
